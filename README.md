# examples-starter

一个基于 `pnpm` `monorepo`、`vue3`、`vite`、多页的快速启动模版，主要用于写一些例子等

## 使用步骤

### 1.安装依赖

```shell
pnpm install
```

项目使用的 `pnpm` 作为包管理工具，如未安装 `pnpm` 的话需要安装 `pnpm`

```shell
npm install -g pnpm
```

### 2.初始化 `package.json`

```shell
pnpm init-pkg
```
